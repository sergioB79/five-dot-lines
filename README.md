# Five-Dot Lines

Solo puzzle de “linhas de 5” (horizontal, vertical, diagonais) com:
- Seed inicial (duas linhas de 8 em cruz grossa)
- Regras sem sobreposição (por eixo)
- Até 2 janelas no mesmo eixo por jogada
- Highscore (localStorage), Show Moves (once)
- Dark mode, som opcional
